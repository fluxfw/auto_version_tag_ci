# auto_version_tag_ci library

## Usage

### `.gitlab-ci.yml`

```yml
include:
  - https://plugins.studer-raimann.ch/Customizing/global/auto_version_tag_ci/build/auto_version_tag_ci.yml
```

### CI variables

Set `AUTO_VERSION_TAG_TOKEN` ci variable

## Requirements
* PHP >=7.0
